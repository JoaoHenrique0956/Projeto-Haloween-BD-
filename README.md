# Projeto-Haloween-BD-
o projeto “Cofre de Doces Criptografado” tem como objetivo armazenar, de forma segura, os tipos de doces coletados durante o halloween. o sistema foi desenvolvido em python, utilizando tkinter para a interface gráfica, pymongo para conexão com o banco de dados mongodb atlas, e fernet  para proteger as informações sensíveis.
